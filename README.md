## Выполнено задание №19: Альтернативная регистрация пользователя в MantisBT через REST API

Реализовано:

* Генерация уникального username и email.
* Создание пользователя на James через REST API.
* Начало регистрации в Mantis через REST API.
* Получение письма с подтверждением и извлечение ссылки.
* Завершение регистрации через ссылку и установка пароля.
* Проверка возможности входа через HTTP.
* Добавлено логирование всех шагов для наглядности.
* Создан вспомогательный класс `RegistrationHelper` для управления процессом регистрации.
* Используется `JamesApiHelper` для работы с James REST API.
* Используется `RestApiHelper` для работы с Mantis REST API.
