### Задание с курса Software Testing
### Задание №6 — Реализовать двухуровневую архитектуру тестового набора
**Цель**

Повторить действия, продемонстрированные в лекциях, и реализовать двухуровневую архитектуру тестового набора.

### Шаги выполнения

1. Создать общий базовый класс ***TestBase***

   * Перенести в него вспомогательные методы, включая:

        * инициализацию тестового окружения;

        * завершение тестов;

        * общие методы, используемые в тестах.

2. Создать класс ***ApplicationManager***

   * Перенести в него вспомогательные методы из тестов.

   * Реализовать логику управления основными компонентами приложения.

3. Выделить специализированные классы-помощники:

   * ***GroupHelper***

   * ***LoginHelper***
   * Перенести в них из ***ApplicationManager*** соответствующие методы, связанные с группами и логином.

4. Создать общий базовый класс для помощников — ***HelperBase***

   * Вынести в него низкоуровневые операции, такие как:

     * заполнение отдельных полей;

     * нажатие на кнопки или ссылки;

     * взаимодействие с элементами интерфейса.
