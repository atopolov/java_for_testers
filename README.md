## Выполнено задание №15: Реализованы тесты для добавления контакта в группу и удаления контакта из группы 

- Действия во всех тестах выполняются через пользовательский интерфейс, а информация, используемая при проверках, загружается из БД.
- При создании теста на удаление контакта из группы были использованы следующие шаги с предусловиями:
1. Если нет группы — создаём её
2. Если в группе нет контакта — создаём контакт и привязываем к группе
3. Удаляем контакт из группы

Проверяем с помощью Assertions:
- количество контактов уменьшилось на 1
- конкретного контакта больше нет в группе
