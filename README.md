Тестовый сценарий для добавления контакта в группу:

1. Создать контакт без группы
2. Убедиться, что контакт не состоит в нужной группе
3. Добавить его в группу через интерфейс:
- выбрать контакт
- выбрать группу в dropdown
4. Нажать кнопку "Add to group"
