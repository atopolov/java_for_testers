## Параметризованные тесты AddressBook (Задание № 9)

Тесты создания **групп** и **контактов** на Java с JUnit 5.

## Особенности
- Генерация тестовых данных через **Faker**:
    - пустые поля
    - только имя
    - полностью случайные значения
- Проверка **увеличения количества элементов** после создания
- Параметризованные тесты через `@ParameterizedTest` и `@MethodSource`

## Структура
- `tests.groups.GroupCreationTests` — тесты для групп
- `tests.contacts.ContactCreationTests` — тесты для контактов
- `model.*Data` и `model.*DataGenerator` — модели и генераторы данных

## Запуск
- Через IDE или Gradle
- Все комбинации данных создаются и проверяются автоматически
