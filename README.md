## Задание №10: Тестирование создания и удаления групп и контактов

## Цель
Проверка корректного создания и удаления групп и контактов.

## Подход
Используются **параметризованные тесты** с различными данными.

## Сравнение списков
- До и после операций создаётся список объектов:
    - `List<GroupData>` для групп
    - `List<ContactsData>` для контактов
- Проверяется, что новые объекты добавлены или удалены корректно.

## Проверка идентификаторов
- У создаваемых групп и контактов обязательно проверяется наличие уникального `id`.
