- Реализована загрузка информации о тестовом стенде из конфигурационного файла

## Задание №12: Реализовать генераторы тестовых данных

- Созданы генераторы данных для групп и контактов с поддержкой форматов **JSON**, **YAML** и **XML**.
- Сгенерированные файлы автоматически используются провайдерами данных в тестах.
- Формат и путь к файлу задаются через параметры командной строки.
- Тесты загружают данные через `@MethodSource`, обеспечивая гибкость и повторное использование входных данных.
